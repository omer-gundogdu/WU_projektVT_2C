Lektion vecka 4

Idag har har börjat brainstorma idéer och kom på idéen om att hjälpa till mina bröder med deras barnvagns shopping företag, just nu har de ingen hemsida.
Det som gick bra idag var att jag kom på idéen men det som var mindre bra var att jag inte kunde göra något med idéen eftersom jag behövde papper samt att jag behövde prata med mina bröder om idéer samt detaljer de ville ha på sin hemsida.

Jag började också med planeringen men det blev strul med github och jag försökte få ihop det, gick inte bra. Saken jag måste göra tills nästa lektion är att få till github att fungera.

Lektion vecka 5

Idag har försökt fixa till github men det gick inte, jag har fått inspiration för min sida men kunde inte skissa eftersom github strulade och tog min tid. Jag behöver fixa till github tills nästa lektion för att den tar för mycket tid att fixa och jag faller bakom i min planering.

Lektion vecka 6

Idag har jag börjat med htlm för första sidan och skrivit css för den. Det som gick bra var att jag fixade github och fick tillgång till github så att jag kan också arbeta hemifrån. Jag hade några css problem men jag sökte upp de och fick de att fungera.

Jag ska börja göra layouten för andra html dokumenter.

Lektion vecka 8

Idag har jag försökt göra min hamburger meny men har misslyckats och därmed gett upp eftersom det tog för mycket tid och jag kunde be om hjälp eller söka hjälp online istället för att fixa till den i skolan. Eftersom jag har det svårt att börja med ett projekt/arbete tycker jag att det är bättre att jobba hemma och kommer därmed jobba mer på eget hand.

Nästa gång ska jag se till att börja med arbetet så tidigt som möjligt för att kunna koncentrera mig och komma igång.

Jag har extendat min navbar till 100% från 80% eftersom den blev bättre. Nästa gång ska jag fixa till sections och börja med att göra en layout för andra sidorna.
